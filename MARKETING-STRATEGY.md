## Competitive Landscape

### Direct Chrome Extension Competitors: NONE

No Chrome extension combines AI-powered boolean query generation + search reports + examiner's reports. This is a unique position.

| Extension | What it does | Threat |
|-----------|-------------|--------|
| Petapator | AI analysis of patents already found | Low |
| Readpatents | Gemini chat on Google Patents ($20/mo) | Low |
| Patent Plus | Summarization & dark mode | Minimal |
| Patent Search Tool | Right-click → Google Patents (no AI) | Minimal |
| BOOL | Generic boolean builder for recruiters | Minimal |

### Indirect Competitors (SaaS)

| Tool | Price | Overlap |
|------|-------|---------|
| Perplexity Patents | Free/$20mo | AI search, no boolean output, no reports |
| PQAI (open source) | Free | Prior art, PDF reports, no boolean |
| PatSeer | $300+/mo | AI + Boolean bridge, enterprise pricing |
| Orbit / PatSnap / Derwent | $5K-$500K/yr | Enterprise platforms |
| InnovationQ Plus | ~$49/user/mo | NL + Boolean, full SaaS |
| DeepIP | $200-500/user/mo | Prior art + drafting copilot |

### Market Gap

- Boolean queries are still required by patent examiners for reproducibility
- Enterprise tools cost $300-$500K/year
- No Chrome extension fills the AI → Boolean → Report pipeline
- Price sweet spot: $10-30/mo sits between free tools and enterprise SaaS

## Fiverr / Freelance Platform Strategy

### Market Opportunity

- Top Fiverr patent search seller (pro_usa): **8,500+ orders at $40+ = $340K+ lifetime revenue**
- Fiverr price range: $5-$335 per search
- Upwork: 168+ open patent search jobs at any time, $50-100/hr rates
- Also active: PeoplePerHour ($70-545), Freelancer.com

### Strategy: Sell the Service, Not the Tool

Fiverr prohibits direct product sales but allows tool-powered services with human review.

#### Recommended Gig: "I will do an AI-powered patent search and prior art report"

| Tier | Price | Deliverable | Delivery |
|------|-------|-------------|----------|
| Basic | $25 | Quick search + list of relevant patents | 1 day |
| Standard | $50 | Deep search + summary report + patentability notes | 2 days |
| Premium | $120 | Full landscape analysis + examiner's report + consultation | 3 days |

#### Gig Extras
- Rush delivery: +$15
- Additional jurisdictions: +$10 each
- Detailed written opinion: +$30

#### Amazon FBA Angle
Separate gig targeting Amazon sellers who need patent/trademark checks before sourcing. High volume, lower complexity, perfect for the tool.

### Why This Works

1. Tool does the heavy lifting — minutes vs hours for manual searchers
2. Human review added (required by Fiverr AI policy) improves quality
3. Undercut $40 baseline while delivering faster
4. Every order = live demo → brand recognition → organic extension installs
5. Dual revenue: service income + Chrome Web Store installs

### The Funnel

```
Fiverr/Upwork orders
  → Clients experience quality/speed
    → Some Google "AI patent search tool"
      → Find extension on Chrome Web Store
        → Direct installs + organic growth
```

### Platform Rules (Important)

- Cannot promote external products/links on Fiverr
- Can mention "proprietary patent search tools" for credibility
- Must disclose AI tool usage when asked
- Must deliver human-reviewed, customized results
- Fiverr takes 20% of earnings
